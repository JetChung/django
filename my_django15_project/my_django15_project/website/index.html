Who's There

An easy solution to see who's at the door

Features

Who's There lets you...


...detect faces

Tells you when someone is at your doorstep

...know

Texts you notifications and a picture if someone is there

...communicate

Tell someone if you're not home, can't answer, or will be there soon

...watch

Check in to see the person live

...capture

Record a video or take a picture for evidence

Open Source!

Write your own program for the Who's There using the included Raspberry Pi
